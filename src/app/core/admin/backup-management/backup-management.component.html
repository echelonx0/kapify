How Backups Work

  • • Automatic: Backups run daily at 02:00 SAST without any action needed
  • • Email Alerts: You'll receive a summary email at admin[at]kapify.africa after each backup
  • • 30-Day Retention: Backups older than 30 days are automatically deleted to save space
  • • Safe & Secure: All backups are encrypted and stored in Supabase Storage
  • • Quick Recovery: Contact support if you need to restore data from a backup
